## Getting Started

```bash
# clone the project
git clone https://github.com/MapGIS/MapGIS-Boot-Docs.git

# enter the project directory
cd MapGIS-Boot-Docs

# install dependency
npm install # Or yarn

# development
npm run docs:dev # OR yarn docs:dev
```
